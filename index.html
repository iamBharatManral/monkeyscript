Monkey Programming Language (Typescript Edition)

The Monkey programming language is a simple, interpreted language designed primarily for educational purposes. It was introduced in Thorsten Ball's book, Writing an Interpreter in Go, which guides readers through building an interpreter for Monkey from scratch.

Key Features

  • Dynamic Typing: Variables in Monkey are dynamically typed, allowing for flexibility in assigning different types of values.
  • First-Class Functions: Functions are first-class citizens in Monkey, meaning they can be assigned to variables, passed as arguments, and returned from other functions.
  • Lexical Scoping: Monkey utilizes lexical scoping rules, where the scope of a variable is determined by its position within the source code.

Basic Syntax and Examples

1. Variables and Data Types

Monkey supports several basic data types, including integers, booleans, strings, arrays, and hashes.

// Integer
let age = 25;

// Boolean
let isStudent = true;

// String
let name = "Monkey programming language";

// Array
let myArray = [1, 2, 3, 4, 5];

// Hash
let person = {"name": "Anna", "age": 25};

2. Functions

Functions in Monkey are declared using the fn keyword and can be assigned to variables.

// Function declaration
let add = fn(a, b) {
    a + b; // Implicit return
};

// Function invocation
let result = add(2, 3); // result is 5

3. Conditionals

Conditional statements in Monkey use the if keyword, optionally followed by an else block.

let number = 10;

if (number > 5) {
    "Greater than 5"; // Implicit return
} else {
    "5 or less"; // Implicit return
}

4. Recursion

Monkey supports recursive function calls, allowing functions to call themselves.

// Define a `fibonacci` function
let fibonacci = fn(x) {
    if (x == 0) {
        0; // Implicit return
    } else {
        if (x == 1) {
            return 1; // Explicit return
        } else {
            fibonacci(x - 1) + fibonacci(x - 2); // Recursion
        }
    }
};

// Function invocation
let result = fibonacci(5); // result is 5
